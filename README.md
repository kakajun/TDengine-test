# ☀️ 光伏数据 AI 查询助手 (Solar AI Bot)

本项目基于 **TDengine** 和 **大语言模型 (LLM)**，让您可以通过自然语言（聊天）的方式查询光伏场站的发电数据，并自动生成图表。

## 🚀 快速开始

### 第一步：环境准备 (最重要！)

1.  **安装 TDengine 数据库**
    *   请务必先阅读并按照 [SETUP_GUIDE.md](SETUP_GUIDE.md) 中的步骤安装 TDengine。
    *   确保数据库服务已启动（端口 6041 可用）。

2.  **验证环境**
    *   运行以下命令检查数据库连接：
    ```bash
    python check_env.py
    ```
    *   如果看到 "✅ 连接成功"，请继续。如果失败，请检查 TDengine 安装。

### 第二步：安装 Python 依赖

在项目根目录下运行：
```bash
pip install -r requirements.txt
```

### 第三步：生成模拟数据

为了让系统有数据可查，我们需要先生成一些模拟的光伏历史数据：
```bash
python data_generator.py
```
*   成功后会提示“数据生成完成”。

### 第四步：启动应用

**推荐启动增强版 (Plus)：**
包含上下文记忆、智能列映射与超限自动处理功能。

```bash
streamlit run plus/app.py
```

*   基础版启动（旧版）：`streamlit run app.py`
*   系统会自动打开浏览器，访问 `http://localhost:8501`。

## 🌟 增强版特性 (Plus)

最新的 `plus/` 版本对系统进行了全面重构与升级，主要特性包括：

1.  **🧠 智能上下文记忆**
    *   **多轮对话**：AI 能够记住您关注的 **站点 (Station)**、**设备 (Turbine)** 和 **时间范围**。
    *   **自动继承**：当您问“查询 F01 的有功功率”后，接着问“无功功率呢？”，系统会自动继承 F01 和之前的时间范围。
    *   **持久化**：上下文设置会自动保存，下次打开页面时无需重复输入。

2.  **🔍 智能列映射 (Smart Mapping)**
    *   **语义匹配**：无需记住复杂的数据库列名。例如，输入“有功功率”或“发电功率”，系统会自动匹配到 `dc` 列。
    *   **海量字段支持**：针对拥有数百个测点的超级表，采用 Top-K 筛选机制，精准提取最相关的列注入 Prompt，避免 Token 浪费。

3.  **🛡️ 上下文超限自动处理**
    *   **滚动摘要**：当对话过长时，系统会自动将早期历史压缩为摘要，确保长时间对话不会“撑爆”模型上下文。
    *   **动态降级**：如果 Prompt 仍然过长，系统会自动精简 Schema 描述，优先保留核心指标。

4.  **⚡ 模块化架构**
    *   代码重构为 `plus/` 模块包，分离 UI、逻辑、数据库与 AI 模块，更易于维护与扩展。

docker start taosdb-ee

## 💡 如何使用

1.  **配置设置** (左侧侧边栏)：
    *   **数据库配置**：默认通常不需要修改 (localhost:6041)。
    *   **AI 模型配置**：输入您的 API Key（如 OpenAI, DeepSeek, 智谱 AI 等）。
        *   如果是 **DeepSeek**：
            *   Base URL 填: `https://api.deepseek.com`
            *   Model Name 填: `deepseek-chat` (或者 `deepseek-reasoner`)
    *   **上下文记忆 (Context Memory)** (新功能)：
        *   在此处可以手动查看或修改当前的 **站点**、**设备列表** 和 **时间范围**。
        *   修改后点击保存，AI 在下一轮对话中将立即应用新的上下文。

2.  **开始提问**：
    *   在对话框输入问题，例如：
        *   "Station_A 今天的总发电量是多少？"
        *   "Station_B 过去 7 天的功率曲线"
        *   "对比 Station_A 和 Station_B 昨天的日发电量"

3.  **查看结果**：
    *   AI 会自动生成 SQL 并查询数据库。
    *   如果是趋势类数据，系统会自动绘制折线图。
